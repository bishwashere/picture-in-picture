## How to run:

1. `docker-compose up`

The server is live at:
http://0.0.0.0:5001/

2. Copy any youtube video from website or app and paste in this URL


Example:

http://0.0.0.0:5001/https://www.youtube.com/watch?v=yMIsVC7ijY8

(from website)

or

http://0.0.0.0:5001/https://youtu.be/D60f8-IOtps

(from app)
